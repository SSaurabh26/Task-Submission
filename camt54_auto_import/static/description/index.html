Requirements

Dependencies

  • Odoo v17.0+
  • account_statement_import_camt (OCA module)
  • Python watchdog library

System Requirements

  • Read/write access to monitored folders
  • Proper file permissions for moving/deleting files
  • Network access if folders are on remote systems

Setup Instructions

  1. Install Dependencies: Install the OCA account_statement_import_camt module
  2. Configure Journals: Set up your bank journals in Odoo
  3. Create Configuration: Go to CAMT54 Auto Import > Configurations and create new configurations
  4. Set Folder Paths: Configure watch folder, processed folder, and error folder paths
  5. Test Connection: Use the "Test Connection" button to verify folder access
  6. Enable Auto Import: The scheduled action runs every 5 minutes by default

Usage

Automatic Import

Once configured, the system will automatically:

  • Monitor the specified folder for new CAMT54 files
  • Import statements into the configured journal
  • Attempt automatic reconciliation based on your settings
  • Move files to appropriate folders based on processing results
  • Log all activities for review and troubleshooting

Manual Import

Use the manual import wizard to:

  • Import individual files outside of the automatic process
  • Override reconciliation settings for specific imports
  • Test imports before setting up automatic processing
